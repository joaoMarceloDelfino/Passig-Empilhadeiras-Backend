## Dependências

Para rodar essa aplicação você precisará de: 

- [PostgreSql 17.6](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)
- [JDK 17](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)

## Rodando a aplicação

1. Duplique `.env.example` e renomeie para `.env`
2. Edite o `.env` conforme necessário
